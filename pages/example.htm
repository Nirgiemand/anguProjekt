Routing

As known, AngularJS is a Single-page application, which means that everytime you click on page elements, AngularJS loads you the exact component for the page and adds it to main "template". Because of this, website functions a lot faster.
Basically, you create one Main index.html, which has all main components of your page - template, and then the "views" which create the variability of website and where all your different substances are.
